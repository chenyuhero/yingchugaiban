盈楚（上海）资产管理有限公司（以下简称“盈楚资产”）


以打造令人信任和尊敬的高端资产管理平台为己任，是经中国证券基金业协会批准的“私募基金管理人”


秉承“诚信、敬业、创新、共赢”的经营理念，盈楚资产以战略家的眼光、金融家的思维、银行家的操守、企业家的责任，汇聚社会资本，驭领时代投资。


公司汇棸了一大批来自金融，投行，法律等各领域的投资精英，风险把控能力强，投资策略稳健而务实，投资眼光精准而独到，是值得信赖的专家型团队。
